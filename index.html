1. Definición de CSS

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje que permite aplicar estilos a las páginas web, controlando el diseño y presentación visual de los documentos HTML. CSS permite separar el contenido de una página (HTML) de su apariencia visual, haciéndola más flexible y fácil de mantener.

2. Funcionamiento Básico de CSS

CSS funciona aplicando reglas de estilo a elementos HTML. Cada regla define cómo deben mostrarse ciertos elementos en la página. Estas reglas se aplican en cascada, lo que significa que se pueden combinar varios estilos y el navegador determina cuál es el estilo final que se aplicará.

3. Incluir CSS en un archivo HTML

Existen tres maneras principales de incluir CSS en un archivo HTML:

  • En línea: añadiendo el estilo directamente dentro de un elemento HTML usando el atributo style.
  • Interno: utilizando una etiqueta <style> en la sección <head> del archivo HTML.
  • Externo: creando un archivo .css separado y enlazándolo a HTML con una etiqueta <link> en la sección <head>.

4. Definición de una regla CSS

Una regla CSS es una instrucción que se aplica a elementos HTML específicos para cambiar su apariencia visual. Cada regla está compuesta por un selector y un bloque de declaración.

5. Elementos de una regla CSS

Cada regla CSS consta de:

  • Selector: define a qué elemento(s) HTML se aplicará la regla.
  • Bloque de declaración: contiene una o más declaraciones de estilo encerradas entre llaves { }.

6. Componentes de una Regla CSS

Selector: Especifica el elemento HTML al que se aplicará la regla. Por ejemplo, p aplicará la regla a todos los párrafos <p> de la página.

Bloque de declaración: El área dentro de las llaves { }, que contiene las propiedades y sus valores para el estilo.

Propiedad: Define qué aspecto del estilo se cambiará (por ejemplo, color o font-size).

Valor: Especifica el ajuste para la propiedad, como blue para color o 16px para font-size.

7. Medios CSS

Los medios CSS permiten definir estilos específicos para diferentes dispositivos o tamaños de pantalla mediante las llamadas media queries. Esto es útil para hacer diseños responsivos, que se ajustan automáticamente a cualquier dispositivo.

8. Comentarios CSS

Los comentarios en CSS permiten añadir notas dentro del código que no afecten el diseño y ayudan a mantenerlo organizado. Para añadir un comentario, se usa /* comentario */.

9. Sintaxis de la definición de las propiedades CSS

Cada propiedad en CSS se define con una estructura de propiedad: valor;. Las propiedades van seguidas de dos puntos (:), luego el valor, y la línea termina con un punto y coma (;). Se pueden combinar varias propiedades en una sola declaración.
